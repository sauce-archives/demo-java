## How to run tests

### 🤖Android native app test
```bash
cd appium-examples
mvn test -Dtest=AndroidNativeAppTest
```

### 🍎iOS native app test
```bash
cd appium-examples
mvn test -Dtest=IOSNativeAppTest
```
### iOS web app test on Emusim
```bash
cd appium-examples
mvn test -Dtest=IOSWebAppExample
```
